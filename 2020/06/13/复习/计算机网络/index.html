通信指标

数据传输速率 R = 1 / T (bps)
传输速率 = 1 / 脉冲带宽(一位的持续时间)
码元速率 C = 1 / t (baud)
码元速率 = 1 / 传输一个码元的时间

码元速率与数据传输速率的关系 R = C log2 M (bps)
M为一个码元所取的离散值个数

奈奎斯特准则 C = 2B (baud)
B为信道带宽
总结 R = 2B log2 M

香农公式 R = B log2 (1 + S / N)
S/N为信号的信噪比 1dB = 10 log10 S/N
误码率 Pe = Ne / N

发送数据F，约定好的校验G
多项式F转化为二进制，同时加上最高次幂个零
G / F  F被除 模二除  加零后的F加上冗余码
发送
G / F  无余数说明无错 有余数证明有错

能检测出 1.全部单个错 2.全部离散的二位错 3.全部奇数个错误 4.全部长度小于或等于k的突发性错误 5.能以 1-(1/2)k-1次方 位的突发性错误

IP

共享介质以太网

星型结构
4个阶段
ALOHA
时隙ALOHA
载波侦听和多路访问CSMA
CSMA/CD 先听后发，边听边发，冲突停止，随即重发
帧长度
帧长度 = （ 2 * 距离 * 速度 ）/ 电速
退避算法
1.确定退避时间T，T=串口长度2t
2.定义重传次数k，小于10时，k=次数。大于10时，k=10。
3.k = 1，0 ~ T之间随机重发
4.k = 2，0 ~ 2T之间随即重发
5.重传16次还冲突，则禁止发送
检查帧长度是否小于64B，小于说明是冲突后的帧
